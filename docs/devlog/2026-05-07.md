## 遇到的问题与解决

1. **golangci-lint v2 配置 schema 改了** — 把 v1 syntax 升级成 v2（formatters 单独分组）
2. **viper 自定义 EnvKeyReplacer 类型签名错误** — 改用 `strings.NewReplacer`
3. **viper Unmarshal 不读 env-only 字段** — 加显式 `BindEnv` 列表
4. **lumberjack 拉取 TLS 超时** — 切到 goproxy.cn
5. **testcontainers v0.42 弃用 `WithWaitStrategyAndDeadline`** — 改用显式 `wait.ForLog` / `wait.ForListeningPort`
6. **prometheus client 把 collectors 移到子包** — 用 `collectors.NewGoCollector()` 等新 API
7. **dev 启动需要 PG 密码** — Makefile run-dev 注入 `AIBAO_POSTGRES_PASSWORD=aibao`
